# 0x1C. C - Makefiles

This project contains examples of Makefiles with varying levels of complexity.

## Files

- `0-Makefile`: Basic Makefile without variables
- `1-Makefile`: Makefile with variables and standard rules
- `2-Makefile`: Makefile with compilation flags and pattern rules
- `3-Makefile`: Makefile with automatic dependency generation

## Usage

To use a specific Makefile:
```bash
make -f <makefile_name>
